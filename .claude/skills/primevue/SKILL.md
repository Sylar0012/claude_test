## PrimeVue 사용 가능한 컴포넌트 목록

아래 목록에 없는 컴포넌트는 사용하지 말 것.
별도 패키지가 필요한 경우 설치 여부를 먼저 확인할 것.

### Form (`@primevue/forms` — 별도 설치 필요)
- Form, FormField

### Input
- InputText, InputNumber, InputMask, InputOtp, InputGroup, InputGroupAddon, InputIcon
- Textarea, Password, Checkbox, CheckboxGroup, RadioButton, RadioButtonGroup
- Select, MultiSelect, CascadeSelect, Listbox, SelectButton, ToggleButton, ToggleSwitch
- DatePicker, ColorPicker, Knob, Rating, Slider, InputChips
- AutoComplete, TreeSelect, Editor

### Button
- Button, ButtonGroup, SplitButton, SpeedDial

### Data
- DataTable, Column, ColumnGroup, Row
- DataView, OrderList, PickList, Paginator
- Tree, TreeTable, Timeline, OrgChart (OrganizationChart)
- VirtualScroller

### Panel
- Accordion, AccordionPanel, AccordionHeader, AccordionContent
- Card, Fieldset, Panel, Divider
- ScrollPanel, Splitter, SplitterPanel
- Tabs, TabList, Tab, TabPanels, TabPanel
- Stepper, StepList, Step, StepItem, StepPanels, StepPanel
- Toolbar

### Overlay
- Dialog, Drawer (Sidebar), ConfirmDialog, ConfirmPopup
- Popover (OverlayPanel), DynamicDialog, Tooltip

### Menu
- Menu, Menubar, MegaMenu, TieredMenu, ContextMenu
- PanelMenu, TabMenu, Breadcrumb, Dock

### Message
- Message (InlineMessage), Toast

### Media
- Image, ImageCompare, Galleria, Carousel

### Misc
- Avatar, AvatarGroup, Badge, OverlayBadge
- BlockUI, Chip, Skeleton, Tag, Terminal
- ProgressBar, ProgressSpinner, MeterGroup
- ScrollTop, FloatLabel, IftaLabel, IconField, Fluid

### Directive
- Tooltip, Ripple, BadgeDirective, StyleClass
- FocusTrap, AnimateOnScroll, KeyFilter

### Service
- ConfirmationService, ToastService, DialogService
